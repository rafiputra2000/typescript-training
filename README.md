# typescript-training
Basic programming typescript for beginners

Typescript

- [ ] Bahasa Typescript di developed ole Microsoft pads tahun 2012.
- [ ] Typescript mendukung OOP programming, seperti Class, Object, Interfaces, Inheritance.
- [ ] Typescript merupakan superset dari javascript dan ECMAscript.
- [ ] Web browser tidak akan mengerti namanya typescript, maka dari itu typescript akan dijadikan javascript tau bisa disebut “transpiling”


Typescript run di VSC
- [ ] tsc nama_file.ts maka file typescript akan transpiling menjadi file javascript kemudian file javascript akan di run menggunakan command line node nama_file.js
- [ ] Catatan utama nama_file.js hasıl transpiling mangan perdah di edit.
- [ ] Pada saat typescript di run menggunakan tsc nama_file code typescript walaupun salah akan meng-generate file JS, solusi yang digunakan adalah menggunakan command tsc —noEmitOnError nama_file 
- [ ] Getter setter pada typescript akan berjalan pada EcmaScript 5 maka untuk menjalankannya menggunakan command tsc —target ES5 —noEmitOnError nama_file


Typescript Class
- [ ] Typescript pada class menggunakan properties, constructor, getter dan setter
- [ ] Pada properties memiliki access modifiers yaitu public, protected, private
- [ ] Public merupakan access modifier yang memiliki sifat dimana properties dapat diakses diluar class, default pembuatan properties yaitu public 
- [ ] Protected merupakan access modifier yang memiliki sifat dimana properties juga tidak dapat diakses diluar class akan tetapi access modifier ini hanya dapat diakses oleh class turunannya saja.
- [ ] Private merupakan access modifier yang memiliki sifat dimana properties hanya data diakses didalam class itu saja.
